# RepeatedFileScanner

> Repeated files scanner


## Usage

```java
-------------------------------------------------------------
rfs 
Repeated files scanner
Author : Maokebing
Version : 1.0
-------------------------------------------------------------
Usage:
       rfs [-fsp] [args]
      -f [file_path]
      -s [size eg. only scan size of file > this arg]
      -d [use delete flag it will del the repeated files]

```
